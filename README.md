# tb-classification 
tuberculosis classification based on x-ray images for Final Project

## Model used
* CNN
* VGG16

## Dataset used
* 1500 Normal
* 1500 Active
* 1500 Laten

## Training scenarios
* 1. 70% Training, 20% Validation, 10% Testing
* 2. 70% Training, 10% Validation, 20% Testing
* 3. 80% Training, 10% Validation, 10% Testing
